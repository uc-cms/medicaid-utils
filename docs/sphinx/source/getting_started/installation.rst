Installation
============

From PyPI
---------

.. code-block:: bash

   pip install medicaid-utils

From Source
-----------

.. code-block:: bash

   git clone https://github.com/uc-cms/medicaid-utils.git
   cd medicaid-utils
   pip install -e .

Requirements
------------

- Python >= 3.11
- Core dependencies: dask, pandas, numpy, pyarrow
- See ``requirements.txt`` for the full dependency list
