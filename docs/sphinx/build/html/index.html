medicaid-utils Documentation

medicaid-utils is a Python toolkit for constructing patient-level analytic files from Medicaid claims data. It implements validated cleaning routines, variable construction methods, and public-domain clinical measure algorithms for both MAX and TAF CMS file formats.

Built on Dask for scalable, distributed processing of large-scale claims datasets.
